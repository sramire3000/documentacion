# OPEN JDK's Ubuntu 24

### Install OPEN JDK 17
```bash
sudo apt -y install openjdk-17-jdk
java --version
```
### Install OPEN JDK 21
```bash
sudo apt install openjdk-21-jdk 
java --version
```
### Install OPEN JDK 25
```bash
sudo apt install openjdk-25-jdk
```

### Cambiar de Jdk
```bash
sudo update-alternatives --config java
```
