# RUST

## Paquetes
- [The Rust community’s crate registry Browse All Crates](https://crates.io/)

### It looks like you’re running macOS, Linux, or another Unix-like OS. To download Rustup and install Rust, run the following in your terminal,
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
### Update Rust
```
rustup update
```

### Version de Rust
```
rustup --version
```

### Version de Cargo
```
cargo --version
```
### Version compilador
```
rustc --version
```
### Compilar un programa
```
rustc nombrePrograma.rs
```

### Crear proyecto con Cargo
```
cargo new hello_cargo
```
### Crear un proyecto tipo libreria
```
cargo new hello_cargo_lib --lib
```

### Build con Cargo
```
cargo build
```

### Compila y ejecuta
```
cargo run
```

### Verificar si compila el proyecto
```
cargo check
```
### Build con optimizaciones (Usuario final)
```
cargo build --release
```
