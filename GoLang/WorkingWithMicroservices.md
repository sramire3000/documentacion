# Curso Working with Microservices in Go


## Crear modulo
```
go mod init broker
```

## Instalar Paquetes en el curso
```
# Es el router principal de Chi.
go get github.com/go-chi/chi/v5

# Un conjunto de middlewares oficiales para Chi
go get github.com/go-chi/chi/v5/middleware

# Un middleware para manejar CORS (Cross-Origin Resource Sharing).
go get github.com/go-chi/cors
```
