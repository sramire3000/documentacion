### 📌 Comandos esenciales de Go
```
go run archivo.go	Compila y ejecuta directamente un archivo Go sin generar binario.
go build	        Compila el código y genera un ejecutable en el directorio actual.
go install	        Compila y guarda el ejecutable en el $GOPATH/bin para uso global.
go mod init	        Inicializa un nuevo módulo Go creando el archivo go.mod.
go mod tidy	        Limpia dependencias no usadas y agrega las faltantes en go.mod.
go get paquete	    Descarga e instala paquetes externos desde repositorios.
go test	            Ejecuta pruebas unitarias definidas en archivos _test.go.
go fmt archivo.go	Formatea el código siguiendo las convenciones de estilo de Go.
go version	        Muestra la versión instalada de Go.
go env	            Muestra variables de entorno relacionadas con Go.
```

### 🔑 Flujo típico de trabajo en Go

1. Inicializar proyecto: go mod init nombre_modulo
2. Escribir código en archivos .go
3. Ejecutar pruebas rápidas: go run main.go
4. Compilar binario: go build
5. Instalar ejecutable globalmente: go install
6. Gestionar dependencias: go get y go mod tidy
7. Probar código: go test
8. Formatear código: go fmt


### 📂 Crear un proyecto

1. mkdir mi_proyecto
2. cd mi_proyecto
3. go mod init ejemplo.com/mi_proyecto
