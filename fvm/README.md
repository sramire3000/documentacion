# Getting Started





# Verificar instalación
```bash
fvm --version
```

# Listar versiones disponibles de Flutter
```bash
fvm releases
```

# Instalar una versión específica de Flutter
```bash
fvm install stable
```

# O instalar la última versión
```bash
fvm install latest
```

# Configurar la versión por defecto
```bash
fvm global stable
```
